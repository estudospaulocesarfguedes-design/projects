# Pizzaria Boa Fatia - Fluxo de Caixa

Este projeto é uma aplicação web para gerenciamento de fluxo de caixa da Pizzaria Boa Fatia. A aplicação permite o registro de receitas e despesas, além de fornecer gráficos e indicadores financeiros.

## Estrutura do Projeto

- `src/index.html`: Documento HTML principal que estrutura a aplicação web, incluindo referências a bibliotecas externas como Chart.js e jsPDF, além de links para arquivos CSS e JavaScript.
- `src/css/style.css`: Arquivo de estilos que define o layout, cores, fontes e outros aspectos visuais da aplicação.
- `src/js/app.js`: Código JavaScript que implementa a funcionalidade da aplicação, incluindo manipulação de eventos, cálculos e interações com os elementos HTML.
- `src/assets/fonts`: Diretório que contém arquivos de fontes utilizados na aplicação, garantindo que o texto seja exibido corretamente de acordo com o design.

## Como Executar a Aplicação Localmente

1. **Clone o repositório**:
   ```bash
   git clone <URL do repositório>
   cd pizzaria-boa-fatia
   ```

2. **Abra o arquivo `index.html` em um navegador**:
   - Você pode abrir o arquivo diretamente no navegador ou usar um servidor local para uma melhor experiência.

## Como Implantar a Aplicação Online

Você pode usar plataformas como GitHub Pages, Netlify ou Vercel para implantar sua aplicação online. Aqui está um breve resumo de como fazer isso:

1. **GitHub Pages**:
   - Faça o push do seu projeto para um repositório no GitHub.
   - Vá para as configurações do repositório e habilite o GitHub Pages na seção "Pages".
   - Escolha o branch (geralmente `main` ou `gh-pages`) e a pasta raiz como fonte.
   - Seu site estará disponível em `https://<username>.github.io/<repository-name>`.

2. **Netlify**:
   - Crie uma conta no Netlify.
   - Arraste e solte sua pasta do projeto (ou conecte seu repositório do GitHub).
   - O Netlify implantará automaticamente seu site e fornecerá uma URL.

3. **Vercel**:
   - Crie uma conta no Vercel.
   - Importe seu projeto do GitHub.
   - O Vercel implantará automaticamente seu site e fornecerá uma URL.

Escolha qualquer uma dessas opções com base na sua preferência por facilidade de uso e recursos.
